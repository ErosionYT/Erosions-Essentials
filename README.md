# Erosion's Essentials
Plugin coded by ErosionYT for public use.

Fixed by [WolfDen133](https://github.com/WolfDen133)

## Features
- [x] Announcement system (can change in the config)
- [x] Easy gamemode control
- [x] Custom join messages
- [x] Freeze commands
- [x] Staff chat
- [x] Brag command
- [ ] Speed command
- [ ] Night vision
- [ ] Anti-Spam
